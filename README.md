## Instalasi :

- Clone Github 
- Install Composer di projek (composer install).
- Copy env (cp .env.example .env).
- Hapus cache "opsional" (php artisan config:cache)
- Generate Key (php artisan key:generate)
- Sambungkan database
- Migrate
